3. Format Invoice Date as DD-MMM-YYYY (e.g. 01-Jan-2026)
4. Add a column "Tax Amount" = Invoice Amount × Tax Rate — format as Accounting
5. Add a column "Total Payable" = Invoice Amount + Tax Amount — format as Accounting
